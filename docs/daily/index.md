## 收录标准

- 一条技巧最好只解决一个明确问题
- 尽量包含真实场景、推荐做法和注意点
- 优先记录会重复遇到、能明显提升效率的内容

## 写作模板

1. 适用场景
2. 推荐做法
3. 示例代码或命令
4. 注意点
5. 延伸阅读或关联技巧

## 使用方式

- 想快速查一个高频技巧时，直接从分类进入。
- 想持续积累时，每条技巧尽量保持“场景 -> 做法 -> 注意点”的结构。
- 后续新增内容时，优先补到现有分类中，保持目录稳定。
