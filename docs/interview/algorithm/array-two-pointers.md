# 数组与双指针

## 高频题型

- 两数之和
- 三数之和
- 盛最多水的容器
- 移除重复元素

## 准备建议

- 先暴力，再优化
- 讲清楚指针移动的依据

## 常见追问

- 为什么排序后更适合双指针
- 什么时候该左右夹逼，什么时候该快慢指针

## 标准回答框架

1. 先说暴力解法
2. 再说优化思路
3. 再解释指针为什么这样移动

## 题库示例

### 问题 1：三数之和为什么通常先排序

**回答要点**

- 排序后更容易利用左右双指针夹逼
- 可以更方便地去重
- 也能让指针移动有明确方向，而不是盲目枚举

**常见追问**

- 去重通常在什么位置做
- 为什么时间复杂度能从三重循环降下来

### 问题 2：快慢指针常见用在哪些题里

**回答要点**

- 链表找中点
- 判断是否有环
- 数组去重或原地移除元素
- 滑动窗口类题目里也可以理解成双指针的变种

**常见追问**

- 左右指针和快慢指针的核心差别是什么
- 什么场景下双指针不适合
