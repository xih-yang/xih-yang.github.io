# 二叉树

## 高频题型

- 前中后序遍历
- 层序遍历
- 二叉树最大深度
- 最近公共祖先

## 常见追问

- 递归和迭代遍历各自的优缺点
- DFS 和 BFS 在树题里分别适合什么场景

## 标准回答框架

1. 先选遍历方式
2. 再说明节点处理顺序
3. 最后补时间复杂度和空间复杂度

## 题库示例

### 问题 1：前中后序遍历分别适合什么场景

**回答要点**

- 前序常用于复制树、序列化等场景
- 中序常用于二叉搜索树相关问题
- 后序常用于先处理子节点再处理父节点的题
- 关键是理解访问根节点的时机不同

**常见追问**

- 为什么后序适合求树高度
- 递归遍历和栈模拟遍历怎么选

### 问题 2：层序遍历和深度优先遍历怎么选

**回答要点**

- 如果题目关注层级关系，通常优先 BFS
- 如果题目更适合递归拆解，DFS 常更自然
- 选择遍历方式要看题目需要的中间信息是什么

**常见追问**

- BFS 的队列里通常存什么
- DFS 一定比 BFS 更省空间吗
