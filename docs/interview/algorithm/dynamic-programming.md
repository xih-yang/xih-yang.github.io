# 动态规划

## 高频题型

- 爬楼梯
- 最大子数组和
- 最长递增子序列
- 零钱兑换

## 常见追问

- 状态定义为什么这么设
- 转移方程是怎么推出来的
- 一维 DP 和二维 DP 怎么判断

## 标准回答框架

1. 先找状态
2. 再找转移
3. 再定初始化和遍历顺序

## 易错点

- 一上来就背公式，没先说状态含义
- 只会做题，不会解释为什么能这样转移

## 题库示例

### 问题 1：动态规划题一般怎么入手

**回答要点**

- 先判断问题是否存在重复子问题和最优子结构
- 再定义状态，明确 `dp[i]` 或 `dp[i][j]` 代表什么
- 再写转移方程、初始化和遍历顺序

**常见追问**

- 怎么快速判断这题适不适合 DP
- 为什么遍历顺序会影响结果

### 问题 2：一维 DP 和二维 DP 怎么判断

**回答要点**

- 看状态需要几个维度来描述
- 如果只跟一个序列位置有关，常见是一维
- 如果和两个位置、区间或两个字符串有关，常见是二维

**常见追问**

- 二维 DP 能不能压缩成一维
- 状态压缩时最容易错在哪里
