# 冲突与协作问题

## 常见问题

- 和同事意见不一致怎么办
- 需求反复变更你怎么处理
- 项目延期时你如何推进

## 回答建议

- 讲事实，不要只讲情绪
- 重点体现沟通、推进和结果

## 推荐结构

1. 先说明冲突背景
2. 再讲你如何沟通和推进
3. 最后讲结果与复盘

## 易错点

- 过度强调对方的问题
- 讲了情绪，但没讲你做了什么

## 题库示例

### 问题 1：和同事意见不一致时你怎么处理

**回答要点**

- 先对齐目标，而不是先争论方案
- 再用数据、风险和成本来讨论取舍
- 如果还无法统一，可以推动小范围验证或找更高层决策

**常见追问**

- 如果对方比你资深怎么办
- 如果最后证明你错了，你怎么复盘
