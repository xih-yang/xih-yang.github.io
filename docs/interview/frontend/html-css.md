# HTML / CSS 面试题

## 高频问题

- 语义化标签有什么价值
- `display: none`、`visibility: hidden`、`opacity: 0` 的区别
- BFC 是什么，什么时候会触发
- Flex 和 Grid 的适用场景
- 响应式布局一般怎么做

## 回答思路

- 不只说定义，要顺带讲实际使用场景
- 布局题最好能给出两三种实现方式并比较

## 常见追问

- 为什么语义化会影响 SEO
- BFC 能解决哪些布局问题
- Flex 和 Grid 选型时你怎么判断

## 标准回答框架

1. 先给概念定义
2. 再说典型场景
3. 再补一个和真实开发相关的例子
4. 最后说常见误区或边界

## 易错点

- 只会背定义，不会举场景
- 布局题只会一种写法，没法比较取舍

## 题库示例

### 问题 1：BFC 是什么

**回答要点**

- BFC 是块级格式化上下文，是一种独立布局区域
- 它可以隔离内部和外部布局影响
- 常见用途包括清除浮动、避免外边距重叠和控制布局边界

**常见追问**

- 哪些 CSS 属性会触发 BFC
- BFC 为什么能解决浮动塌陷
