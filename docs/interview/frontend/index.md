# 前端面试

前端面试通常覆盖基础、框架、工程化、性能、安全和项目实践。

## 已整理专题

- [HTML / CSS 面试题](./html-css.md)
- [JavaScript 面试题](./javascript.md)
- [Vue 面试题](./vue.md)
- [React 面试题](./react.md)
- [前端工程化面试题](./engineering.md)

## 准备建议

- 先抓高频基础题，再补框架与项目题。
- 每一类问题都尽量准备“定义 + 原理 + 实战例子”。

## 回答重点

- 基础题讲清楚原理和场景
- 框架题讲清楚状态、渲染和工程边界
- 项目题讲清楚你的真实职责和取舍
