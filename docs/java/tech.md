# JAVA 技术

## JAVA 基础

### 基本概念

JAVA 是一种面向对象的编程语言，由 Sun Microsystems 公司于 1995 年推出。它具有平台无关性、面向对象、分布式、多线程等特性。

### 核心特性

- **面向对象**：封装、继承、多态
- **平台无关性**：一次编写，到处运行
- **自动内存管理**：垃圾回收机制
- **多线程**：支持并发编程
- **异常处理**：提供完善的异常处理机制

## JAVA 集合

### 常用集合类

- **List**：有序集合，可重复元素
  - `ArrayList`：基于数组实现，随机访问快
  - `LinkedList`：基于链表实现，插入删除快
- **Set**：无序集合，不可重复元素
  - `HashSet`：基于哈希表实现
  - `TreeSet`：基于红黑树实现，有序
- **Map**：键值对集合
  - `HashMap`：基于哈希表实现
  - `TreeMap`：基于红黑树实现，有序
  - `ConcurrentHashMap`：线程安全的哈希表

## Spring 框架

### Spring 核心

- **控制反转 (IoC)**：将对象创建和依赖管理交给 Spring 容器
- **依赖注入 (DI)**：通过构造函数、setter 方法或字段注入依赖
- **面向切面编程 (AOP)**：将横切关注点与业务逻辑分离

### Spring Boot

Spring Boot 是 Spring 框架的简化版，提供了自动配置、嵌入式服务器等特性，使开发更加快速和便捷。

## 数据库

### MySQL

MySQL 是一种关系型数据库管理系统，广泛应用于 Web 应用中。

### Redis

Redis 是一种内存数据库，常用于缓存、会话存储等场景。

## 多线程

### 线程创建

- 继承 `Thread` 类
- 实现 `Runnable` 接口
- 实现 `Callable` 接口

### 线程同步

- `synchronized` 关键字
- `Lock` 接口
- `Atomic` 类
- `Concurrent` 集合

## 设计模式

- **创建型模式**：单例模式、工厂模式、抽象工厂模式
- **结构型模式**：适配器模式、装饰器模式、代理模式
- **行为型模式**：观察者模式、策略模式、模板方法模式

## 最佳实践

- **代码规范**：遵循阿里巴巴 JAVA 开发规范
- **异常处理**：合理处理异常，避免空捕获
- **资源管理**：使用 try-with-resources 管理资源
- **性能优化**：合理使用缓存、减少数据库查询
- **安全**：防止 SQL 注入、XSS 攻击等
