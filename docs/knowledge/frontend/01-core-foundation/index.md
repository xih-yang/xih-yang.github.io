# 前端核心基础

这一层解决的是“页面为什么能工作”。结构、样式、脚本、浏览器环境和原生能力，是后续框架与工程化的底座。

## 学习重点

- 先把语义化、布局、响应式和 DOM 操作练熟。
- 再把闭包、原型链、异步、模块化和面向对象串起来。
- 最后回头看 jQuery 与原生封装，理解框架出现前后分别解决了什么问题。
