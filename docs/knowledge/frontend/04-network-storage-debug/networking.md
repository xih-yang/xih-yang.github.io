# 网络请求与实时通信

网络层要解决的是请求封装、错误处理、跨域、重试和实时消息同步。

## 核心范围

- axios：实例封装、拦截器、统一错误处理、取消请求
- 跨域：同源策略、CORS、代理、JSONP 的边界
- WebSocket：长连接、心跳、断线重连、消息协议设计
- SSE：单向实时推送、浏览器支持与使用场景

## 实战关注点

- 把鉴权、刷新令牌和异常提示放进统一网络层
- 根据业务选择轮询、SSE 还是 WebSocket

## 常见问题

- 请求封装只包了一层函数，没有处理鉴权和异常分类
- 不了解同源策略和 CORS，排查跨域问题效率低
- 实时通信只会连上，不会做心跳、重连和状态恢复

## 学习路线

1. 先掌握基础请求封装和错误处理。
2. 再理解跨域、代理和浏览器安全边界。
3. 最后补 WebSocket、SSE 和实时消息场景。
