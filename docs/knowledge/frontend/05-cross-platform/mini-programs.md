# 小程序生态

小程序是很多前端团队进入跨端的第一站，重点在于平台能力、包体限制和端能力差异。

## 核心范围

- 微信小程序原生：页面、组件、生命周期、分包、性能与审核
- Taro：多端编译、React 风格开发和端差异处理
- uni-app / uni-app x：Vue 风格开发、跨端能力和生态插件
- 云开发：数据库、云函数、存储与鉴权配套

## 实战关注点

- 关注包体积、分包策略和平台审核限制
- 多端编译时要预留端差异处理能力
- 登录、支付、分享等平台能力要单独梳理
