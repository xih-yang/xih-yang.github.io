# 原生客户端

理解原生客户端不是要求前端一定转原生，而是帮助我们更好地和移动端团队协作。

## 核心范围

- Android：Kotlin、Activity / Fragment、Jetpack 基础认知
- iOS：Swift / Objective-C、ViewController、导航与生命周期
- 前端协作重点：接口约定、WebView 容器通信、混合架构边界

## 协作重点

- 明确 H5、容器和原生页面之间的职责边界
- 提前对齐桥接协议、登录态和埋点方案
- 对平台审核、签名和发版流程保持基本认知
