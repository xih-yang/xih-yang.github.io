# 前端安全

前端安全不是只有“防 XSS”这一个点，而是输入、输出、存储、传输和权限校验的整体协作。

## 核心范围

- XSS / CSRF 防护思路和常见误区
- 加解密与签名的基础认知
- 接口安全：鉴权、签名、重放防护、权限边界
- 合规：隐私弹窗、数据脱敏、日志和埋点合规

## 实战关注点

- 前端输入、服务端校验和网关策略要配合起来看
- 安全方案要兼顾体验、成本和风险等级
- 涉及用户数据时，合规要求要前置考虑

## 常见问题

- 只背 XSS / CSRF 定义，不知道在真实项目怎么防
- 把“前端加密”当成核心安全手段
- 埋点、日志和错误上报里暴露了敏感信息
