## 学习顺序建议

1. 先打牢页面基础：HTML、CSS、布局、语义化、响应式、原生 JS。
2. 再学习框架生态：React、Vue、SSR/SSG，以及状态管理和路由。
3. 然后补工程化能力：TypeScript、构建工具、包管理、规范、测试。
4. 接着进入生产实践：网络、鉴权、调试、性能、安全、监控。
5. 最后扩展跨端、全栈、部署、架构与 AI 应用能力。

## 使用方式

- 想系统学习时，从九大模块总览页往下看。
- 想快速查漏补缺时，直接进入具体主题页。
- 后续扩展时，优先往既有模块追加子页，尽量不打乱一级目录。
