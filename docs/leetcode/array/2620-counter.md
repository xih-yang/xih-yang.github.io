# 2619. 数组原型对象的最后一个元素

- 难度：简单
- 类型：数组 / 原型 / 基础实现

## 题目

请你编写一段代码实现一个数组方法，使任何数组都可以调用 `array.last()` 方法，这个方法将返回数组最后一个元素。如果数组中没有元素，则返回 `-1`。

你可以假设数组是 `JSON.parse` 的输出结果。

## 示例

### 示例 1

```text
输入：nums = [null, {}, 3]
输出：3
解释：调用 nums.last() 后返回最后一个元素 3。
```

### 示例 2

```text
输入：nums = []
输出：-1
解释：因为数组为空，所以返回 -1。
```

## 思路

这道题本质上是在 `Array.prototype` 上补一个方法。

在方法内部，`this` 就是当前调用它的数组，所以我们只需要判断两件事：

1. 数组是否为空
2. 如果不为空，返回下标 `this.length - 1` 对应的元素

整个过程不需要遍历，直接取最后一个位置即可。

## 代码

```javascript
Array.prototype.last = function () {
  if (this.length === 0) {
    return -1;
  }

  return this[this.length - 1];
};
```

## 复杂度

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`

## 容易出错的点

- 不要写成 `this[this.length]`，因为数组最后一个元素的下标是 `length - 1`。
- 空数组时要显式返回 `-1`，不能返回 `undefined`。
- 这里是在原型对象上挂方法，所以函数定义要保证能访问到调用时的 `this`。

## 小结

这题不难，但很适合作为原型方法题的入门。核心不是语法，而是要明确：

- `this` 指向当前数组
- 空数组的边界要先处理
- 最后一个元素的位置永远是 `length - 1`
