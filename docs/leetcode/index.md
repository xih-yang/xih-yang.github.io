## 解题技巧

1. **理解题目**：仔细阅读题目，理解问题要求
2. **分析示例**：通过示例理解输入输出
3. **选择算法**：根据题目类型选择合适的算法
4. **编写代码**：实现算法，注意边界条件
5. **测试用例**：测试各种情况，确保代码正确
6. **优化性能**：分析时间和空间复杂度，优化代码

## 写题建议

- 先写最直接的正确解，再考虑是否需要优化。
- 每道题至少补一句“为什么这样做是对的”，避免只记代码。
- 同类型题目尽量放在同一目录，方便复习时横向对比。
